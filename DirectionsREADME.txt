Directions:

Add the file to to the data folder and save it in the format "ggYEAR.json", (where the year is put the year of the awards) and then run main with the year of the file in the command prompt.
So you would run "py main.py 2016" etc. in the command prompt of in the path of our repository.
The packages we downloaded were nltk , numpy, and imdb.py. 

You can download imdb py from https://imdbpy.sourceforge.io/ or download the file in the Requirements file
You can install numpy using "pip install numpy" in the command prompt or download it manually from https://pypi.org/project/numpy/ or  download the file in the Requirements file
and you can install nltk using "pip install nltk" in the command prompt or downlaod it manually from https://pypi.org/project/nltk/ or download the file in the Requirements file
